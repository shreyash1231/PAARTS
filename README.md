# Street Food Stalls - Dhule

This project is a simple website that lists street food stalls in Dhule city. It provides a user-friendly interface where visitors can explore different food stalls and their offerings in the city. The website is built using HTML, CSS, and JavaScript.

## Features

- **Food Stalls Listings**: View a list of food stalls in Dhule city.
- **Search Functionality**: Search for specific food stalls or dishes.
- **Responsive Design**: The website is designed to be mobile-friendly and responsive.

## Tech Stack

- **HTML**: For the structure of the web pages.
- **CSS**: For styling the web pages.
- **JavaScript**: For adding interactive features, such as search functionality.

## How to Run

1. Download or clone the repository.
2. Open the `index.html` file in a web browser.

Alternatively, you can host the project on a web server for live access.
